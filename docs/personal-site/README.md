# Darri Eythorsson - Personal Website

Personal academic website for Darri Eythorsson, Earth-system modeling researcher at University of Calgary.

## Features

- Clean, academic design
- Responsive layout
- Dark/light mode support
- Research projects showcase
- Publications list
- Professional links

## Deployment

This site is designed to be hosted on GitHub Pages through the Eythorsson Earth System Modeling organization.

## Local Development

Simply open `index.html` in a web browser to view locally.
